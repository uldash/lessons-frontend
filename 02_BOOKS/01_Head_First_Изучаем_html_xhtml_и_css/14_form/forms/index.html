Элемент <input> типа radio создает переключатели, которые являются взаимоисключающими. Это похоже на старые переключатели в машинах: если вы выбираете одно положение, то все остальные становятся неактивными.

Элемент <input> типа checkbox создает флажки, которые могут быть либо установленными, либо снятыми. Вы можете использовать сразу несколько флажков, при этом можно установить любое их количество.



Элемент <textarea> создает многострочное текстовое поле для ввода данных. Если вы вводите больше текста, чем может вме ститься в одну строку, то справа появляется полоса для прокрутки.

Элемент <select> создает на веб-странице раскрывающийся список. Он предоставляет возможность выбирать предлагаемые варианты ответов.Элемент <select> работает в паре с элементом <option>, создающим меню.

Чтобы создать меню, используйте элемент <option> в паре с элементом <select>. Задавайте элемент <option> для каждого пункта меню.

Элементы формы HTML5

Элемент <input> типа number вводит ограничение, согласно которому пользователи смогут вводить в соответствующее поле только числа. Вы можете задать минимальное и максимальное число, которое можно будет ввести, посредством опциональных атрибутов.

Элемент <input> типа range аналогичен элементу <input> типа number, за исключени ем того, что он обеспечивает отображение ползунка вместо поля ввода.

Используйте элемент <input> типа color для задания цвета. При щелчке мышью на соответствующем элементе управления на экране появится палитра цветов, из которой вы сможете выбрать требуемый цвет, вместо того чтобы вручную вводить его название или значение.


Используйте элемент <input> типа date для задания дат,применяя такой элемент управления, как календарь. Данный элемент управления генерирует строку допусти мого формата даты для отправки серверному сценарию.
